### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、画师、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 画师
5. 画师作品
6. 系统公告
7. 后台管理
8. 个人中心
9. 我的收藏

**画师功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 画师作品管理
6. 约稿信息管理
7. 稿件作品管理



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 作品类型管理
4. 用户管理
5. 画师管理
6. 画师作品管理
7. 约稿信息管理
8. 稿件作品管理
9. 系统管理（关于我们、系统简介、轮播图管理、系统公告）
